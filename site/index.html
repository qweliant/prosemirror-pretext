🎨

Canvas rendering

Text drawn straight to canvas via Pretext's pure-arithmetic line breaking. No reflow, no DOM text nodes, no contenteditable quirks.

🌊

Float-wrap

Drag an image and the paragraphs wrap around it — the same obstacle-carving Pretext's own demos use, wired to real document nodes.

🦻

Accessibility mirror

A hidden, screen-reader-visible DOM mirror of the doc, a labelled textbox, live-region announcements, and reduced-motion support.

📐

Real ProseMirror

Genuine PM document model, transactions, schema, marks, history. We just replaced prosemirror-view.

📝

Batteries

Headings, marks, links, nested lists, blockquote, code blocks, images, gap cursor, alignment, rich paste, undo/redo.

⚡

Fast layout

A Y-independent layout cache + per-block re-segmentation means typing only re-measures the run that changed.

quick start 🐣

it's a peer of the ProseMirror stack — bring your own schema

# install (peer deps: prosemirror-{model,state,commands,keymap,gapcursor,schema-list})
bun add prosemirror-pretext

// then — canvas, all the way down 🐸
import { CanvasEditor, markSpecs, buildMarkKeymap } from 'prosemirror-pretext'
import { EditorState } from 'prosemirror-state'

await document.fonts.ready // fonts must be loaded before layout

const editor = new CanvasEditor({
  state: EditorState.create({ doc, schema }),
  container: document.querySelector('#editor'),
  ariaLabel: 'My notes',
})
